## 🚀 Featured Projects

<details>
<summary><strong>🔬 NeuroLens — Multimodal Medical Image Analysis System</strong></summary>
<br/>

An end-to-end deep learning pipeline for automated analysis of medical imaging data (MRI, CT, X-Ray) integrating vision-language models for clinical report generation. Built with a focus on explainability, precision, and deployment-readiness in resource-constrained environments.

| Attribute | Details |
|-----------|---------|
| **Stack** | PyTorch, CLIP, BioGPT, FastAPI, Docker, PostgreSQL |
| **Scale** | 50K+ medical images across 6 modalities |
| **Performance** | 94.2% diagnostic accuracy, 38ms inference latency |
| **Security** | HIPAA-compliant data handling, encrypted inference pipeline |
| **Impact** | Reduced radiologist review time by ~40% in pilot study |
| **Repository** | [github.com/akshitakushwaha/neurolens](#) |

Designed a custom attention-weighted fusion architecture combining ResNet-50 vision encoders with BioGPT language decoders. Implemented GradCAM-based visual explanations for each prediction to support clinical interpretability. The system was containerized with Docker and exposed through a RESTful FastAPI interface with async inference support.

</details>

<details>
<summary><strong>🧾 LexiQ — Legal Document Intelligence Engine</strong></summary>
<br/>

A retrieval-augmented generation (RAG) system for legal document comprehension, clause extraction, and contract risk scoring. Enables lawyers and compliance teams to query large document corpora in natural language with citation-grounded responses.

| Attribute | Details |
|-----------|---------|
| **Stack** | LangChain, OpenAI GPT-4, FAISS, FastAPI, Redis, React |
| **Scale** | Processes 10,000+ page legal corpora with sub-second retrieval |
| **Performance** | 91% answer faithfulness on legal QA benchmarks |
| **Security** | Role-based access control, encrypted vector store, audit logging |
| **Impact** | Reduced document review time by 65% in enterprise trials |
| **Repository** | [github.com/akshitakushwaha/lexiq](#) |

Engineered a hybrid retrieval pipeline combining dense semantic search (FAISS + OpenAI embeddings) with BM25 sparse retrieval. Implemented a re-ranking layer using cross-encoder models to improve answer grounding. Designed a clause taxonomy and trained a fine-tuned classifier for automated clause-type tagging across 18 categories.

</details>

<details>
<summary><strong>🎙️ VocalSync — Real-Time Multilingual Speech Translation</strong></summary>
<br/>

A streaming speech-to-speech translation system supporting 12 Indian and global languages. Combines ASR, neural machine translation, and TTS into a unified low-latency pipeline designed for accessibility and cross-linguistic communication.

| Attribute | Details |
|-----------|---------|
| **Stack** | Whisper, mBART-50, SpeechT5, PyTorch, WebSockets, FastAPI |
| **Scale** | Supports 12 languages, 500+ concurrent stream sessions |
| **Performance** | End-to-end latency under 900ms, 88 BLEU on translation |
| **Security** | Ephemeral audio processing, no persistent storage of voice data |
| **Impact** | Deployed in 3 community accessibility programs |
| **Repository** | [github.com/akshitakushwaha/vocalsync](#) |

Architected a streaming audio pipeline using WebSocket connections to process audio chunks in real-time. Integrated OpenAI Whisper for transcription, fine-tuned mBART-50 for domain-specific translation, and Microsoft SpeechT5 for natural-sounding synthesis. Optimized inference using ONNX runtime quantization achieving 3.2x speedup over baseline.

</details>

<details>
<summary><strong>📊 FinSentinel — Financial Market Sentiment & Anomaly Detection</strong></summary>
<br/>

An intelligent financial intelligence platform that fuses NLP-driven sentiment analysis of news, filings, and social media with statistical anomaly detection on market time-series to surface early-warning signals for equity movements.

| Attribute | Details |
|-----------|---------|
| **Stack** | FinBERT, LSTM, Kafka, Apache Spark, Grafana, PostgreSQL |
| **Scale** | Ingests 50K+ news articles and 1M+ data points daily |
| **Performance** | 82% precision on directional equity movement signals |
| **Security** | Isolated data ingestion layers, encrypted API keys vault |
| **Impact** | Used by 2 fintech research teams for alpha signal discovery |
| **Repository** | [github.com/akshitakushwaha/finsentinel](#) |

Built a real-time event-driven architecture using Apache Kafka for stream ingestion and Spark Structured Streaming for distributed feature engineering. Fine-tuned FinBERT on domain-specific financial corpora for sentiment scoring. Designed an LSTM-based anomaly detector trained on rolling 90-day windows with adaptive thresholding. Visualized signals through Grafana dashboards with configurable alert logic.

</details>
